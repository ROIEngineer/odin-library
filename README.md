
## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone or download the project files
2. Ensure both `index.html` and `styles.css` are in the same directory
3. Open `index.html` in your web browser

## Usage

1. Open the application in your browser
2. Click the "Add Book" button to open the book form
3. Fill in the book details:
   - Title (required)
   - Author (required)
   - Number of pages (optional)
   - Read status (checkbox)
4. Click "Add" to save the book to your library
5. Click "Cancel" to close the form without adding a book

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- Uses semantic HTML for accessibility
- Responsive design with CSS Flexbox
- Unique book IDs generated using `crypto.randomUUID()`
- Form validation for required fields

## Browser Compatibility

Compatible with all modern browsers that support:
- ES6 JavaScript features
- CSS Flexbox
- HTML5 form validation
